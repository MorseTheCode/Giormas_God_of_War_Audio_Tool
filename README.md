## Features

- Open SBP and BNK files
- View contained WEM entries by ID
- Extract all WEM files from an SBP
- Replace individual WEM files
- Batch replace WEMs by ID from a folder
- Save selected or all modified SBP files

## Usage

- Open one or more `.sbp` / `.bnk` files
- Select an SBP to extract all WEMs
- Select a WEM entry to replace it
- For batch replacement, provide a folder containing `.wem` files named by WEM ID
- Save the modified SBP files

## Notes

- Replacement preserves container structure
- WEM size is fixed per entry
- No audio conversion is performed
